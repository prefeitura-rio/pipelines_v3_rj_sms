# Extração

Pasta para flows de extração, com nomes prefixados por "Extração:".

Flows de extração obtêm dados que não estão prontos para consumo e requerem algum processamento; em seguida, na maioria dos casos, fazem upload para o datalake do dado agora estruturado. Normalmente requerem soluções específicas para o dado sendo extraído. Exemplos incluem interações com APIs, web scraping, extração de arquivos com formatos usados por um único provedor, ...
